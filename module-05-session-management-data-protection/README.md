# Module 5: Session Management and Data Protection

## Objective

Cover session security (cookies, tokens, JWT best practices), data encryption and secure storage, and secure API development and protection against API abuses.

## Contents

- **slides.md** — Session slides (Reveal.js/compatible).
- **demos/** — Demo scripts and prompts.
- **exercises/** — Hands-on exercises.

## Prerequisites

Modules 1–4. Secure coding and auth (Module 4) are assumed.
