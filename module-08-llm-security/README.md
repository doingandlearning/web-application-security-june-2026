# Module 8: LLM and AI-Assisted Coding Security

## Objective

Enable developers to use LLMs and AI-assisted tools (e.g. GitHub Copilot) safely: understand OWASP LLM Top 10 risks, apply trust-but-verify, protect PII and sensitive data, and recognise and mitigate hallucinations and misuse.

## Contents

- **slides.md** — Session slides (Reveal.js/compatible).
- **demos/** — Demo scripts (e.g. Copilot, prompt injection, data leakage).
- **exercises/** — Hands-on exercises.

## Prerequisites

Modules 1–7. No prior LLM experience required; focus is security and safe usage.
