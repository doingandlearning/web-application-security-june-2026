# Module 3: Understanding Common Security Vulnerabilities

## Objective

Introduce the OWASP Top 10, link them to common coding flaws, and give hands-on practice exploiting insecure applications (e.g. DVWA, WebGoat).

## Contents

- **slides.md** — Session slides (Reveal.js/compatible).
- **demos/** — Demo scripts and prompts.
- **exercises/** — Hands-on exercises (DVWA/WebGoat).

## Prerequisites

Modules 1 and 2. Lab environment (DVWA/WebGoat) must be available.
