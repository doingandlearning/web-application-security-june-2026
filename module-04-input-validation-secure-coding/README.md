# Module 4: Input Validation and Secure Coding Practices

## Objective

Prevent SQL injection, cross-site scripting (XSS), and command injection; apply best practices for input sanitization and validation; implement secure authentication and authorization.

- **slides.md** — Session slides (Reveal.js/compatible).
- **demos/** — Demo scripts and prompts.
- **exercises/** — Hands-on exercises.

Modules 1–3. Understanding of common vulnerabilities (Module 3) is assumed.
