# Module 6: Security Testing and Vulnerability Assessment

## Objective

Use OWASP ZAP and Burp Suite for security testing; introduce static and dynamic application security testing (SAST/DAST); cover penetration testing fundamentals for developers.

## Contents

- **slides.md** — Session slides (Reveal.js/compatible).
- **demos/** — Demo scripts and prompts.
- **exercises/** — Hands-on exercises (ZAP, Burp).

## Prerequisites

Modules 1–5. Lab environment with ZAP and/or Burp available.
